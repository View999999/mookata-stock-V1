# 🔥 ระบบสต็อกร้านหมูกระทะ

ระบบจัดการสต็อกสำหรับร้านหมูกระทะบุฟเฟต์ แจ้งเตือนผ่าน LINE Notify

## ฟีเจอร์
- ✅ เช็คของ 2 รอบ: สั่งรอบเช้า + ปิดร้าน
- ✅ สรุปยอดใช้ไปจริง: (ปิดเมื่อวาน + สั่งเช้า) − ปิดวันนี้
- ✅ สั่งของแยกโซน เลือกร้านค้าได้
- ✅ ส่งรายงานผ่าน LINE Notify
- ✅ โหมดเจ้าของ (รหัส 1234) / พนักงาน แยกสิทธิ์
- ✅ บันทึกข้อมูลใน localStorage ไม่ต้องมี server

## วิธีตั้งค่า LINE Notify Token
1. ไปที่ https://notify.line.me
2. Login ด้วยบัญชี LINE
3. My page → Generate token
4. ตั้งชื่อ → เลือกกลุ่มไลน์ → Copy token
5. เปิดแอป → เข้าโหมดเจ้าของ (รหัส 1234) → LINE Token → วาง token

## วิธีเปลี่ยนรหัสเจ้าของ
แก้ไขไฟล์ `src/data.js` บรรทัด `export const OWNER_PIN = "1234"`

## Deploy บน Vercel
1. Push โค้ดขึ้น GitHub
2. ไปที่ vercel.com → Import repository
3. กด Deploy — เสร็จเลย!
